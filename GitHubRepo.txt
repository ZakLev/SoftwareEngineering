An android emulaor that emulates Nintendo Switch funcionality. The developers have their discord and allow users to contact them with questions.

https://github.com/github/gitignore.git - 
Allows the user to easily implement the function to ignore certain files when pushing to a repo, in order to only get the uncompiled raw code.
